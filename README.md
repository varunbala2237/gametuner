## About
Game Tuner is an Android tool that lets you adjust device performance settings with the help of **Shizuku**.  
It is designed mainly for testing and experimenting with system-level configurations.

## Features
- Change device resolution and DPI
- Limit background apps to reduce memory usage
- Enable or disable memory cleaner
- Toggle force GPU rendering
- Apply all settings quickly with a single switch
